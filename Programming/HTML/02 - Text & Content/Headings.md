# Headings `<h1>` – `<h6>`

## Description
Headings define titles and subtitles. There are 6 levels.
`<h1>` is the most important, `<h6>` the least.

## Syntax
```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
```

## Example
```html
<h1>Main Page Title</h1>
  <h2>Chapter 1</h2>
    <h3>Section 1.1</h3>
    <h3>Section 1.2</h3>
  <h2>Chapter 2</h2>
    <h3>Section 2.1</h3>
```

## Notes
- Use only **one `<h1>`** per page (SEO best practice)
- Don't skip levels (e.g. h1 → h3)
- Headings affect **SEO** and **accessibility**
