# `<hr>` – Horizontal Rule

## Description
Inserts a horizontal line, typically used to separate content sections.

## Syntax
```html
<hr>
```

## Example
```html
<h2>Section One</h2>
<p>Content of section one.</p>

<hr>

<h2>Section Two</h2>
<p>Content of section two.</p>
```

## Styled Example
```html
<hr style="border: 2px solid #333; width: 50%;">
```

## Notes
- Self-closing tag
- Semantic meaning: a **thematic break** in content
- Default style is a thin grey line (can be overridden with CSS)
