# `<br>` – Line Break

## Description
Inserts a single line break inside text. Self-closing tag (no closing tag needed).

## Syntax
```html
<br>
```

## Example
```html
<p>
  Line one<br>
  Line two<br>
  Line three
</p>

<!-- Address example -->
<address>
  John Doe<br>
  123 Main Street<br>
  New York, NY 10001
</address>
```

## Notes
- Self-closing: `<br>` or `<br />` (both valid)
- Don't use `<br>` for spacing between paragraphs — use CSS margin instead
- Good use case: addresses, poems, lyrics
