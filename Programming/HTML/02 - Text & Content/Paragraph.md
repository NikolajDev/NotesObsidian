# `<p>` – Paragraph

## Description
Defines a block of text as a paragraph. Browsers add spacing above and below automatically.

## Syntax
```html
<p>Your text here.</p>
```

## Example
```html
<p>This is the first paragraph. It contains some text.</p>
<p>This is the second paragraph. Browsers add space between them.</p>

<p>You can also use <strong>bold</strong> and <em>italic</em> inside paragraphs.</p>
```

## Notes
- Block-level element (takes full width)
- Automatically adds margin top and bottom
- Nesting block elements (like `<div>`) inside `<p>` is invalid
