# HTML – Overview

HTML (HyperText Markup Language) is the standard language for creating web pages.
The browser reads HTML tags and renders them as visual content.

## How it works
1. You write HTML in a `.html` file
2. The browser reads and renders it
3. Tags describe the **structure** of content

## Key Concepts
- **Tags** – building blocks: `<p>`, `<h1>`, `<div>`
- **Attributes** – extra info inside tags: `class`, `id`, `href`
- **Nesting** – tags inside tags
- **Semantic HTML** – using the right tag for the right content
