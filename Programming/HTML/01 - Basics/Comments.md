# HTML Comments

## Description
Comments are notes in the code that the browser ignores and does not display.

## Syntax
```html
<!-- This is a comment -->
```

## Example
```html
<!DOCTYPE html>
<html lang="en">
<body>

  <!-- Main navigation starts here -->
  <nav>
    <a href="/">Home</a>
  </nav>

  <!-- TODO: add footer later -->

</body>
</html>
```

## Notes
- Comments are not visible in the browser
- Useful for explaining code or temporarily disabling elements
- Can span multiple lines:
```html
<!--
  This is a
  multi-line comment
-->
```
