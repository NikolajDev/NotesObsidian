# `<!DOCTYPE html>`

## Description
Tells the browser which version of HTML to use.
Always placed on the **first line** of every HTML file.

## Syntax
```html
<!DOCTYPE html>
```

## Full Example
```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My Page</title>
  </head>
  <body>
    <p>Hello!</p>
  </body>
</html>
```

## Notes
- Not a tag — it's a declaration
- Without it, browser goes into "quirks mode" (older rendering)
- `<!DOCTYPE html>` = HTML5 (current standard)
