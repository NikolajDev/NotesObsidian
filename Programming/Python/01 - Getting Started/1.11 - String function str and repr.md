# 1.11 str() and repr()

## Description
Both functions convert objects into strings, but for different audiences:
- `str()`: Returns a readable, user-friendly string representation of the object.
- `repr()`: Returns an unambiguous, developer-friendly string representation (often valid Python code that can recreate the object).

## Code
```python
import datetime

now = datetime.datetime.now()
print("STR:", str(now))
print("REPR:", repr(now))
```
