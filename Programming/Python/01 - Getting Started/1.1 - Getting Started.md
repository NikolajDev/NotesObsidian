# 1.1 Getting Started

## Description
Python is a high-level, interpreted programming language. 
The `print()` function outputs data to the standard output device (usually your screen).
You can run Python code from a file or directly from the command line.

## Code
```python
print("Hello, this is my first Python script!")
print("Let's start programming.")
```
