# 1.3 Block Indentation

## Description
Where other programming languages use curly brackets `{}` to define a block of code, Python uses indentation (whitespace at the beginning of a line). 
The standard practice is to use 4 spaces per indentation level.

## Code
```python
def check_password(entered_password):
    secret_password = "open_sesame"
    if entered_password == secret_password:
        result = "Access granted"
        print("Welcome inside!")
        return result
    else:
        result = "Incorrect password"
        print("Try again.")
        return result

print(check_password("open_sesame"))
```
