# 1.4 Datatypes

## Description
Variables can store data of different types, and different types can do different things.
Common types include `str` (text), `int` (integers), `float` (decimals), and `bool` (booleans).
You can use functions like `int()`, `float()`, or `str()` to convert data from one type to another (type casting).

## Code
```python
release_year = '2023'
print("Type of release_year:", type(release_year))

year_number = int(release_year)
print("New type:", type(year_number))
print("Year a decade later:", year_number + 10)
```
