# 1.8 Built in Modules

## Description
A module is a file containing a set of functions you want to include in your application. 
Python comes with a large standard library of built-in modules (like `math` or `random`) that you can bring into your code using the `import` statement.

## Code
```python
import math
import random

radius = 5
circle_area = math.pi * math.pow(radius, 2)
print(f"Area of circle with radius 5 is: {circle_area:.2f}")

lucky_number = random.randint(1, 100)
print(f"Lucky number: {lucky_number}")
```
