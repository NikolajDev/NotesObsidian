# 1.5 Collection Types

## Description
Python provides several built-in collection types to store multiple items in a single variable:
- **List:** Ordered and mutable. Allows duplicate members. Defined with `[]`.
- **Dictionary:** Ordered (since Python 3.7) and mutable. Stores `key: value` pairs. Defined with `{}`.
- **Set:** Unordered, unindexed, and unchangeable. No duplicate members. Defined with `{}`.

## Code
```python
# List
favorite_games = ['Witcher', 'Cyberpunk', 'Mario']
favorite_games.append("Zelda")
print("Games:", favorite_games)
```
<div style="background-color: black; color: #00FF00; padding: 10px; font-family: monospace;">
Games: ['Witcher', 'Cyberpunk', 'Mario', 'Zelda']
</div>

```python
# Dictionary
config = {'resolution': '1920x1080', 'vsync': False, 'fps': 60}
print("Target FPS:", config['fps'])
```
<div style="background-color: black; color: #00FF00; padding: 10px; font-family: monospace;">
Target FPS: 60
</div>

```python
# Set
user_ids = {101, 102, 101, 103, 104, 102}
print("Unique IDs:", user_ids)
```
<div style="background-color: black; color: #00FF00; padding: 10px; font-family: monospace;">
Unique IDs: {104, 101, 102, 103}
</div>
