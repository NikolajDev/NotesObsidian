# 1.2 Variables

## Description
Variables are containers for storing data values. 
Python has no command for declaring a variable; it is created the moment you first assign a value to it. 
Variables can even change type after they have been set (dynamic typing).

## Code
```python
player_score = 150
health = 85.5
hero = 'Arthur'
is_alive = True
weapon = None

print(player_score, health, hero, is_alive, weapon)
```

</div>

# Multiple assignment
