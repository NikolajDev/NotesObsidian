# 1.7 User Input

## Description
The `input()` function allows you to pause the program and prompt the user for input. 
Whatever the user types is returned as a string (`str`), which can then be assigned to a variable and manipulated.

## Code
```python
color = input("Enter your favorite color: ")
print(f"Your color is {color}!")
```
