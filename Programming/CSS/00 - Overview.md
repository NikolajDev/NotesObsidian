# CSS – Overview

CSS (Cascading Style Sheets) is the language used to style HTML elements.
It controls colors, fonts, spacing, layout, animations, and more.

## How it works
1. You write CSS rules targeting HTML elements
2. The browser applies the styles and renders the page
3. CSS **cascades** — later rules override earlier ones

## Key Concepts
- **Selector** – targets the HTML element
- **Property** – what you want to change
- **Value** – how you want to change it
- **Cascade** – order and specificity determine which rules win
- **Inheritance** – some properties pass down to child elements
