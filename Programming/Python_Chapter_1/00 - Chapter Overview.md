# Chapter 1: Getting started with Python Language

Python is a widely used high-level programming language for general-purpose programming, created by Guido van Rossum and first released in 1991.

## Core Characteristics
- **Dynamic Type System**: No need to declare variable types.
- **Automatic Memory Management**: Garbage collection handles memory.
- **Multiple Paradigms**: Supports object-oriented, imperative, and functional programming.

## Checking your installation
To confirm that Python is installed correctly, use your terminal.
